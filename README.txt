taasceneview
============

author:  Thomas Atwood (tatwood.net)
date:    2011
license: see UNLICENSE.txt

This is a utility for viewing taascene files.

Usage
=====
    taasceneview <taascene path>

Building
========

## Linux ###
The the following dependencies are required to build on Linux:
    ../taamath
    ../taascene
    ../taasdk
    -lGL
    -lm
    -lrt
    -lX11

### Windows ###
The the following dependencies are required to build on Microsoft Windows:
    ../taamath
    ../taascene
    ../taasdk
    OpenGL32.lib
